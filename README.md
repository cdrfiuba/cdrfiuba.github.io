# cdrfiuba.github.io
La página web del Club de Robótica, publicada en [clubderobotica.com.ar](http://clubderobotica.com.ar).

## Desarrollo local

Instalar [Ruby](https://www.ruby-lang.org/en/downloads/) y [Bundler](https://bundler.io/), y ejecutar:

```bash
bundle install
bundle exec jekyll serve
```

Los cambios se reflejan automáticamente en la carpeta `_site` (ignorada en el repo).
